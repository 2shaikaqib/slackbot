# slackbot
Hello, This is my little slack bot named Revered Snake. This is a pretty basic slack bot. Although it was very complicated to make, lol. It took me at least 3 hours half of which couldn't be captured by hackatime. This slack bot runs on the hack club workspace and is only seen by the person who summons it. It has 4 commands, Pong, Cat fact, Cat image and help. Ping justs says pong, cat fact says a interesting fact about cats, cat image gives you the URL of a cat image, and help says what each of these commands are for. They all use a different API. It runs 24/7 from my laptop.
